# 用户操作手册

## 登录

1. 打开系统首页。
2. 输入用户名和密码，点击「登录」。
3. 演示账号：

   `make backend-seed-demo` 仅会自动创建管理员账号 `demo_admin`（密码 `demo123`，角色 `admin`）。
   其他角色账号需由管理员在「用户管理」页面手动创建，或将下方账号作为创建示例：

   - 管理员：`demo_admin` / `demo123`（脚本自动创建）
   - 财务经理：`demo_finance` / `demo123`（需手动创建，角色 `finance_manager`）
   - 普通查看者：`demo_viewer` / `demo123`（需手动创建，角色 `viewer`）

   > 生产环境切勿使用上述弱密码，请通过 `INIT_PASSWORD` 或用户管理接口设置强密码。

## 财务报告

- 在「财务报告」页面查看已生成的报告列表。
- 点击报告标题查看详情、指标图表和导出选项。
- 填写标题、类型、年份、周期后点击「创建报告」生成新报告。
- 报告状态说明：
  - `pending`：已提交，等待后台处理
  - `processing`：后台生成中
  - `reviewing`：待复核，需人工审批
  - `approved`：已通过，可导出使用
  - `rejected`：已驳回，需重新生成
  - `published`：已发布
  - `failed`：生成失败，可查看错误信息后重试
  - `draft`：草稿

## 文档解析

- 进入「文档解析」页面。
- 选择 CSV / Excel / PDF 文件，点击「上传并解析」。
- 系统后台解析完成后，可在列表查看状态和结构化结果。

## 智能问答

- 进入「智能问答」页面。
- 输入自然语言问题，例如「2025 年 Q1 净利润是多少」。
- 系统返回 SQL 和查询结果。

## 人工审批

- 具有审批权限的用户可在「人工审批」页面查看待审批报告。
- 填写备注后点击「通过」或「驳回」。

## 审计日志

- 管理员和审计员可在「审计日志」页面查看所有关键操作记录。
